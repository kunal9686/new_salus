# **App Name**: SalusWell

## Core Features:

- Landing Page: Display title and quote with a smooth transition to login.
- Authentication: Secure login using email and password with UI.
- Profile Page: Display and manage user profile information.
- Journal Page: Private digital journal for daily entries and reflections.
- Chatbot Page: AI-powered chatbot for personalized wellness guidance; this tool can use profile and journal entries to inform its guidance.
- Inspire Page: Curated inspirational content (quotes, images, videos) based on user preferences.
- Communicate with Others: Facilitate community interaction through messaging or forum features.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to convey calm and clarity.
- Background color: Very light blue (#F0F8FF), almost white, to enhance the minimalist design.
- Accent color: Lavender (#BDB2FF) to add a subtle touch of elegance and serenity.
- Headline font: 'Belleza' (sans-serif) for headings and titles, paired with 'Alegreya' (serif) for body text. Note: currently only Google Fonts are supported.
- Use minimalist icons with a thin stroke to match the modern design.
- Employ a clean and spacious layout with ample white space to promote calmness.
- Incorporate subtle and smooth transitions between pages and sections for a seamless user experience.
